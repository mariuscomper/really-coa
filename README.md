# „Really, coa'?" — Ce dezvăluie două cuvinte despre cum vorbim cu mașinile

O analiză multi-disciplinară a unui post de Facebook despre **code-switching** cu un LLM — privită prin lentilele sociolingvisticii, psihanalizei, antropologiei, psihologiei evoluționiste, teoriei critice, neuroștiinței și filosofiei minții.

## Despre ce e vorba

Un moment aparent banal — cineva îi spune unui chatbot *„Really, coa'?"* — devine punctul de plecare pentru o explorare în profunzime a relației dintre oameni și inteligența artificială. Eseul analizează cum **două cuvinte** dezvăluie mecanisme complexe de identitate lingvistică, intimitate digitală și granițele difuze dintre om și mașină.
